# Nvim Cheat Sheat

## Navigation

<Ctrl-o> - go back to previous location
<Ctrl-i> - go forward to the next location

## Debugging

<F9> - Toggle breakpoint
<F5> - Start Debugging
<F5> - Continue until break
<F10> - Step over
<F11> - Step into
<F12> - Complete the current function
<F3> - Stop debugging
